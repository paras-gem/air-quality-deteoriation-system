# 🌍 AIRQ — India Air Quality Prediction System

A full-stack AI-powered web app that predicts AQI for major Indian cities using a **Random Forest** ML model.

---

## 📁 Project Structure

```
AIRQ/
├── Model/
│   ├── aqi predictor.py            ← Training script (run this first!)
│   ├── AQI_Prediction_India_1.ipynb
│   └── exports/                    ← Auto-generated by training script
│       ├── rf_1h_model.pkl         ← 1-hour prediction model
│       ├── rf_24h_model.pkl        ← 24-hour prediction model
│       └── metadata.json           ← Feature names + accuracy scores
│
├── backend/
│   ├── app.py                      ← Flask API server
│   └── requirements.txt            ← Python dependencies
│
└── src/                            ← React TypeScript Frontend
    └── pages/
```

---

## 🚀 Setup Instructions

### Step 1 — Train the Model
> Place `india_aqi_v6_2024-2025.csv` inside the `Model/` folder first.

```bash
cd Model
pip install pandas scikit-learn matplotlib seaborn joblib
python "aqi predictor.py"
```
This generates `rf_1h_model.pkl`, `rf_24h_model.pkl`, and `metadata.json` inside `Model/exports/`.

---

### Step 2 — Start the Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```
API live at: `http://localhost:5000`

---

### Step 3 — Start the Frontend

```bash
npm install
npm run dev
```

## 🔌 API

**POST** `/predict`

```json
{
  "city": "Delhi",
  "current_aqi": 85,
  "aqi_1h_ago": 82,
  "aqi_24h_ago": 90,
  "temperature": 30.0,
  "precipitation": 0.0,
  "wind_speed": 15.5,
  "hour": 14,
  "month": 3
}
```

**Response:**
```json
{
  "city": "Delhi",
  "predicted_1h": 87.3,
  "predicted_24h": 91.0,
  "category_1h": "Satisfactory",
  "category_24h": "Satisfactory"
}
```
